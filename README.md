## Features

### **Item Buying, Selling, and Lending System**
- Users can list items for sale or lending (tools, books, chargers, calculators, etc.)
- Lendees can send lending requests for specific durations
- Item owners can review and accept/reject lending requests
- Complete transaction management for both sales and rentals

### **Real-time Notifications**
- Instant alerts for lending request approvals
- Notifications when items are sold or reserved
- System updates and activity alerts

### **Flexible Pickup Scheduling**
- Buyers and sellers can mutually agree on pickup times
- Customizable campus locations for item exchange
- Calendar integration for scheduling convenience
- Ensures smooth, hassle-free transactions

### **Smart Search & Filtering**
- Search-based browsing with instant results
- Category-based filtering (Electronics, Clothing, Stationery, Books, etc.)
- Advanced filters: price range, condition, location, availability
- Easy navigation for finding specific items

### **Save for Later / Watchlist**
- Bookmark favorite items for future reference
- Track price changes and availability status
- Compare options before making purchasing decisions
- Personalized collections for each user

### **Transaction History & User Ratings**
- Complete record of all buy/sell/lend transactions
- User rating system after completed transactions
- Builds trust and community reputation
- Review system for item quality and seller reliability

### **User Profiles & Management**
- Personalized user dashboards
- Profile customization with bio and preferences
- Track listed items, saved items, and transaction history
- Admin panel for user management and content moderation

### **Image Uploads & Galleries**
- Multiple image uploads for each listing
- Image preview and gallery display
- Secure file storage and management

### **Responsive Design**
- Mobile-friendly interface for on-the-go access
- Optimized for all screen sizes
- Fast loading and intuitive navigation
