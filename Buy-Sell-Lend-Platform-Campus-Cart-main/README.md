# Campus Cart

## Description
**Campus Cart** is a dedicated platform for university students to **Buy**, **Sell**, and **Lend** items exclusively within their campus community.

## Features
- **Item Buying, selling, and Lending system**
    - Allow Users to list items they want to sell/lend out (e.g., tools, books, chargers, calculators).
    - Lendees can send a lending request for items for a set duration.
    - The user who is interested in lending can accept the requests.
  
- **Notifications**
    - Real-time alerts for lending request approvals and sold notifications.

- **Flexible pick up time and location scheduling**  
    - Both buyer and seller can set a mutually agreed pickup time and location anywhere on campus.
    - Ensures a smooth and hassle-free transaction.
 
- **Search-based Browsing and category-based filtering**  
    - Browse items by searching in the search box.
    - Filter by catagories(e.g., electronics, clothes, stationary) for easies searching.

- **Save for later option**
    - Users can bookmark favorite items to revisit later if the item is available.
    - Helps compare options before making a buying decision.

- **Transaction History and user rating**
    - Keeps a record of past buy, sell, and lend for reference.
    - Users can rate others after transactions to build trust in the community.
